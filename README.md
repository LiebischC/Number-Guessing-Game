# 🔢 Number Guessing Game

A C# console application where the computer selects a random number and the player tries to guess it.  
This project demonstrates the use of the **Random class**, loops, conditionals, and user input handling in C#.

---

## ✨ Features
- Generates a random number between 1 and 100
- Player guesses until the correct number is found
- Feedback after each guess (*Too High* / *Too Low*)
- Tracks the number of guesses
- Option to play again after winning

---

## 📂 Project Structure
- `NumberGuessingGame.sln` — Solution file for Visual Studio
- `NumberGuessingGame.csproj` — Project file with build configuration
